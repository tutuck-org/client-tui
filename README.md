# TuTuck TUI

> [!NOTE] 
> TuTuck is in very early stage. Report any bugs and issues via GitHub issues. If you know Go and like the project - feel free to contribute, we really need contributors

## What's this?
A Text-based User Interface client for connecting to any [tutuck server](https://github.com/tutuck-org/server/). 

TuTuck is a messenger in terminal. Simple. Secure. Works via ssh.

## How do I try it?
First, you need a [server](https://github.com/tutuck-org/server/). You can self host it or ask a friend. Download latest server binaries for your OS in [Releases](https://github.com/tutuck-org/server/releases/latest/)

Once you got a server to connect to, install the client.

Download it from [Releases](https://github.com/tutuck-org/client-tui/releases/latest/), connect to the server... Done! Explore
